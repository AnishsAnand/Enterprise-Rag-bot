## How to Add These Prompts

### In the Admin Panel (Where you are now):

1. **Click "+ Add a banner"** or **"+ New Prompt"**

2. **For each prompt above**:
   - **Title**: Copy the title (e.g., "Create a Kubernetes cluster")
   - **Content**: Copy the prompt text
   - **Optional**: Add tags like `kubernetes`, `deployment`, `security`, etc.

3. **Save** each prompt

4. **Reorder** them by dragging (most common at top)

## How to Customize Further

### Add Your Own Prompts:
1. Think about **common questions** your users ask
2. Create prompts that **save time** for repetitive tasks
3. Include prompts for **troubleshooting** common issues
4. Add **learning resources** for new users

### Tips:
- Keep prompts **concise** but **specific**
- Use **action verbs** (Create, Show, Help, Deploy)
- Include **context** (datacenter, region, resource type)
- Group similar prompts with **tags**
